# survey_practice

## 사용방법
1. npm install
2. npm start
3. [http://localhost:3000/?id=0&name=홍길동](http://localhost:3000/?id=0&name=홍길동) 접속합니다

## 폴더구조 (src 디렉토리 기준)
- __tests__: 유틸함수 테스트 코드
- components: 페이지를 구성하기 위한 컴포넌트 구성
- datas: 설문 진행을 위한 예시 데이터
- pages: 페이지
- static: image 파일
- styles: 공통 css 설정
- utils: 연산 및 데이터 추출을 위한 유틸함수
